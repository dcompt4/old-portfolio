Circularity (Circle Poem)

Circularity is based off of an old project I did when I was in Operation Spark, a high school learning program meant to teach students with little to know experience in coding the core concept of computer programming.

The key differences in this project is that I used the p5 library to code it from scratch myself using randomness and the libraries draw function to constanly update the canvas. I also added a control panel form which allows the viewer to change some aspects of the canvas as wells as some other neat controls.
